#### 异或高级操作
x^0 = x
x^ 1s = ~s
x^(~x) = 1s
x^x = 0
c = a^b => a^c = b,b^c = a
a^b^c = a^(b^c) = (a^b)^c

### 某些移位的操作
1. 将 x 的最右边n位清零： x&(~0 << n )
2. 获取x 的第n位的值： (x>>n) &1
3. 获取x 的第n位的幂值： x&(1<<(n-1))
4. 将第n位置1：x|(1<< n )
5. 讲第n位置置0：x&(~(1<< n ))
6. x 最高位到第n位清零： x&(( 1<< n ) -1 )
7. x 第n位到第0 清零： x & ( ~ ((1<<(n+1)) -1 ))

#### 
x&（x-1） => 清零最后的1
